# JSON-CDN
A Simple Linux CDN Server that exposes a Directory Listing in JSON Format.

---

## 📦 Installation

Clone the Repository:
```bash
git clone https://github.com/PixelCattt/JSON-CDN.git
```

Install Dependencies:
```bash
npm install
```


## ⚙️ Configuration

Set your Root-Path at the Top of ```cdn.js```:
```bash
const ROOT = "/YOUR-PATH-HERE";
```


## 🌍 Usage

Start the CDN-Server:
```bash
npm start
```
