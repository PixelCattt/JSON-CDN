# JSON-CDN
A Simple Linux CDN Server in JSON Format.

---

## 🚀 Features

- Browse Folders with URL Paths
- Open Raw Files (Videos, Images, etc.)
- JSON based Directory Listings
- No Database Required
- Lightweight and Fast

---

## 📦 Installation

Clone the Repository:
```bash
git clone https://github.com/PixelCattt/JSON-CDN.git
```

Install Dependencies:
```bash
npm install
```


## ⚙️ Configuration

Set your Root-Path at the Top of ```cdn.js```:
```bash
const ROOT = "/YOUR-PATH-HERE";
```


## 🌍 Usage

Start the CDN-Server:
```bash
npm start
```
